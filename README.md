# Система управления заявками

Веб-приложение для приёма и обработки заявок от клиентов. Включает форму подачи заявки и панель управления для администраторов.

---

## Возможности

- **Форма заявки** — пользователь заполняет контактные данные и отправляет заявку
- **Панель управления** — просмотр всех заявок, фильтрация по статусу, обновление и удаление
- **Статусы заявок** — «Новая» → «В обработке» → «Завершена»
- **Заметки и результат звонка** — менеджер может вносить комментарии при обработке
- **Валидация данных** — на фронтенде и бэкенде (телефон, email, обязательные поля)

## Установка и запуск

### Предварительные требования

- Python 3.8+
- MongoDB
- Браузер

### 1. Клонируйте репозиторий

```bash
git clone <url-репозитория>
cd project
```

### 2. Запустите бэкенд

```bash
chmod +x start-backend.sh
./start-backend.sh
```

Скрипт автоматически:
- Создаст виртуальное окружение `venv`
- Установит зависимости из `requirements.txt`
- Запустит Flask на порту **6767**

### 3. Запустите фронтенд

В отдельном терминале:

```bash
chmod +x start-frontend.sh
./start-frontend.sh
```

Фронтенд будет доступен на порту **8000**.

### 4. Откройте в браузере

| Страница | URL |
|----------|-----|
| Форма заявки | http://localhost:8000 |
| Панель управления | http://localhost:8000/admin.html |

---

## 🔧 Конфигурация

Создайте файл `.env` в папке `backend/`:

```env
MONGODB_URI=mongodb://localhost:27017/praktika-DB
```

## API

Базовый URL: `http://localhost:6767/api`

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| `GET` | `/applications` | Получить все заявки |
| `POST` | `/applications` | Создать новую заявку |
| `GET` | `/applications/:id` | Получить заявку по ID |
| `PUT` | `/applications/:id` | Обновить заявку (статус, заметка, результат звонка) |
| `DELETE` | `/applications/:id` | Удалить заявку |
| `GET` | `/health` | Проверка работоспособности сервера |

## Валидация

**Телефон:** формат `+7XXXXXXXXXX` или `8XXXXXXXXXX`

**Email:** стандартный формат `name@domain.tld`

**Обязательные поля:** телефон, email, фамилия, имя, отчество, организация
